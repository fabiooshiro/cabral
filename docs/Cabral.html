fillFile(el, fileName)

Pretend to fill an input type file.
The javascript can't change a value of an input with type "file".
So Cabral creates a hidden field to send your file name value.
This value will be intercepted by TomateGrailsPlugin.groovy with a modified version of request.getFile('yourfile.txt') that get that file from <yourapp>/test/resources/yourfile.txt
Parameters:
Name Type Description
el element input file tag
fileName string name of your file
Source:
  • cabral.js, line 193
Example
<input type="file" id="myfile" />

var el = $('#myfile')[0];
cabral.fillFile(el, 'yourfile.txt') ;
